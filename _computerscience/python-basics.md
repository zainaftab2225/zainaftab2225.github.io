### Printing

Output any expression to the screen using the **print()** function.


```python
print("Hello World") # Hello World
print("The answer of 2+2 is: ", 2+2) # The answer of 2+2 is: 4
print("The answer of 2+2 is: ", "2+2") # The answer is: 2+2
```
